Прописка НОК

Что такое прописка для НОК, зачем она нужна и как получить регистрацию для подачи документов на независимую оценку квалификации.

Определение

Что такое «прописка» для НОК

Под пропиской для НОК понимается регистрация по месту жительства или пребывания, данные которой обязательно указываются при подаче документов на экзамен.

Зачем нужна

Для подачи заявки в системе АИС НОК (exam.nostroy.ru, aisok.ru) обязательно указание адреса регистрации. Без этих данных заявление не может быть отправлено. Также требуется копия паспорта с отметкой о регистрации.

Правовая основа

Согласно документам ЦОК, для прохождения НОК требуется копия паспорта (главная страница и страница с пропиской). Это стандартное требование для всех специальностей: строители, проектировщики, изыскатели, ОПБ.

Важно знать

Без прописки сдать НОК нельзя. ЦОК может отказать в допуске к экзамену при отсутствии данных о регистрации. Однако подойдёт и временная регистрация --- она оформляется бесплатно и за 3-8 рабочих дней.
