Нужна ли прописка для сдачи экзамена НОК?

Разбираем требования к регистрации при прохождении независимой оценки квалификации

Краткий ответ

Нет, для сдачи экзамена НОК необходимо иметь хотя бы временную регистрацию.

Обоснование требования

1. Требования к документам для НОК включают:

  • Копию паспорта (2-3 и 5 страницы)
  • Где 5 страница содержит информацию о регистрации по месту жительства

2. Техническая необходимость:

  • В автоматизированных информационных системах НОК (АИС НОК, exam.nostroy.ru, aisok.ru) поле "Адрес регистрации" является обязательным для заполнения
  • При регистрации в системе требуется указать данные прописки из паспорта

3. Правовая основа:

  • Согласно документам ЦОК, для прохождения НОК требуется "паспорт главная страница и страница с пропиской"
  • Это стандартное требование для всех специальностей (строители, проектировщики, изыскатели, пожарная безопасность)

Что делать если нет постоянной прописки

✅ Необходимо оформить временную регистрацию:

  • По месту фактического пребывания
  • Минимальный срок - на период сдачи экзамена
  • Оформляется в МФЦ, паспортном столе или через Госуслуги
  • Временная регистрация бесплатна (госпошлина не взимается)

Документы для временной регистрации:

  • Паспорт гражданина РФ
  • Заявление о регистрации по месту пребывания
  • Согласие собственника жилья
  • Документ-основание (договор аренды, найма и т.п.)

Важные моменты

  • Без указания адреса регистрации в системе невозможно завершить подачу заявки на НОК
  • ЦОК может отказать в допуске к экзамену при отсутствии данных о регистрации
  • Временная регистрация должна быть действующей на момент сдачи экзамена

Практические рекомендации

  1. Оформить временную регистрацию до подачи документов в ЦОК
  2. Приложить копию свидетельства о временной регистрации к пакету документов
  3. Указать адрес временной регистрации при заполнении заявления в АИС НОК
